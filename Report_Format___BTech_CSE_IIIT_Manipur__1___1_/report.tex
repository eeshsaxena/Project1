% ── ABSTRACT ──────────────────────────────────────────────────────
\begin{abstract}
A recurring failure mode of Retrieval-Augmented Generation systems is that they
pass conflicting facts to the language model and let it decide which to believe.
On medical and legal queries this produces unreliable answers: the model may
select a decade-old guideline over a recent one because both chunks score equally
under cosine similarity, which carries no temporal information.

This project builds TruthfulRAG v5, a Knowledge-Graph pipeline that resolves
this conflict before the LLM is called. Starting from the KG-RAG v4 baseline
(arXiv:2511.10375), nine contributions are added. The core innovation is an
exponential temporal decay ($e^{-\lambda t}$, $\lambda=0.08$) that demotes older
facts at retrieval time and a score-gap threshold that discards the stale side
of a conflict automatically. Complementary additions include a year-per-triple
storage scheme in Neo4j, cross-document corroboration scoring, hybrid
BM25+semantic retrieval with Reciprocal Rank Fusion, a three-factor confidence
score, a human-readable audit trail, and automatic schema inference that removes
the last manual configuration step from v4.

On the \texttt{pminervini/HaluEval} benchmark (200 samples), TruthfulRAG v5
rejects 92.0\,\% of hallucinated answers against 77.5\,\% for standard
LangChain RAG. The Conflict Detection Rate doubles from 25\,\% to 50\,\% over
v4. The system runs entirely
locally on a laptop using Qwen2.5-7B-Instruct and Neo4j Community Edition,
with no cloud dependency and no data leaving the machine.
\end{abstract}
