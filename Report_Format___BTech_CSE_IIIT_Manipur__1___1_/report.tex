% ── ABSTRACT ──────────────────────────────────────────────────────
\begin{abstract}
Large Language Models (LLMs) exhibit hallucination and factual inconsistency when their
parametric knowledge conflicts with user-provided documents --- a problem that standard
Retrieval-Augmented Generation (RAG) pipelines do not address. This project implements
and extends TruthfulRAG v5, a Knowledge-Graph (KG) based RAG pipeline that explicitly
detects and resolves temporal and factual conflicts present in document corpora.

Building upon the three-module architecture (Module A: Graph Construction; Module B:
Graph Retrieval; Module C: Conflict Resolution) of TruthfulRAG v4 (arXiv:2511.10375),
version 5 introduces six novel contributions: \textbf{[N1]} cross-document corroboration
scoring that weights facts by how many independent sources agree; \textbf{[N2]}
exponential temporal decay ($e^{-\lambda t}$) ensuring newer facts are preferred over
older ones; \textbf{[N3]} hybrid BM25 and semantic retrieval fused by Reciprocal Rank
Fusion (RRF) for robust document selection; \textbf{[N4]} a human-readable explanation
chain that makes every conflict resolution decision fully auditable; \textbf{[N5]} a
calibrated three-factor confidence score (entropy change, corroboration, temporal
recency); and \textbf{[N6]} a claim verification mode returning structured
\texttt{SUPPORTED}, \texttt{REFUTED}, or \texttt{UNCERTAIN} verdicts with evidence.

A pre-pipeline schema inference step \textbf{[A4+]} renders the system fully
domain-agnostic --- no manual configuration is required when switching between medical,
legal, scientific, or historical corpora. The pipeline runs entirely on local hardware
using the Qwen2.5-7B-Instruct language model served via Ollama and Neo4j Community
Edition as the graph database. A production-quality dual-panel web interface enables
real-time, side-by-side comparison of standard LangChain RAG against TruthfulRAG v5 on
any user-supplied corpus with a single click.

Experimental evaluation across four domain corpora (medical, legal, Indian science, and
space science) demonstrates correct contradiction resolution, a 35\,\% LLM cache hit
rate, and calibrated confidence scores accurately reflecting the degree of evidential
conflict in each domain.
\end{abstract}
